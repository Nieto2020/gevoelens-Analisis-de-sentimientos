
**Gevoelens ==> Analizador de Sentimiento de Texto con Dashboard Web**
---
Geovelens es un prototipo de algoritmo y Web diseñado para detectar y clasificar comentarios en base a "sentiment", pudiendo clasificar el comentario como:

>*BUENO*
>*MALO*
>*NEUTRAL*

---

Aplicación web local que reciba textos (reseñas de productos, tweets o comentarios), procese los datos usando Python para determinar si el impacto es positivo, negativo o neutral, y muestre métricas y gráficas básicas en el navegador.
